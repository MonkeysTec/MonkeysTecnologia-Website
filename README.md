# Monkeys Tecnologia

Site institucional da Monkeys Tecnologia, criado com React, TypeScript e Vite.

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Deploy na Vercel

A Vercel detecta Vite automaticamente. O comando de build e a pasta de saida ja ficam no padrao:

- Build command: `npm run build`
- Output directory: `dist`

## Estrutura principal

- `src/App.tsx`: conteudo e secoes da landing page.
- `src/App.css`: layout, responsividade e identidade visual.
- `src/assets/monkeys-hero.png`: imagem principal do hero.
